# Live-Video-Transmission-and-Reception-Using-mmWave-A-Real-Time-SDR-Based-Demonstration
This project demonstrates a real-time live video transmission and reception system over a millimeter-wave (mmWave) wireless link using an SDR-based testbed. It achieves end-to-end video capture, encoding, mmWave transmission, reception, decoding, and live playback with low latency and high throughput.
